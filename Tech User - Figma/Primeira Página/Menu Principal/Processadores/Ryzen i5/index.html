Como é feita a Classificação ?

A classificação das CPU's é feita com base notas totais.
A nota da CPU leva em conta suas especificações
dívida pela média total do grupo.

Oque é uma CPU ?

CPU - Central Processing Unit (Unidade Central de Processamento)
A CPU é um chip responsável pelo processamento de dados , agindo
como um 'cérebro' para um computador , executando os cálculos.
Um único CPU pode ter vários Núcleos e Threads , que auxiliam o CPU no processamento dos dados.

- Formula -

X / M = R --> V = R * 6

X = pontuação
M = média
R = resultado
V = nota
